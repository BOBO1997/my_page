# ボーカロイドについて

ひとまず完成(2019/09/05)

どうぞ温かい目で

# YouTubeとニコニコ動画

youtubeは2008年くらいからずっと見ていたのですが、その頃のニコニコ動画はアカウントがないと入れない状態で、個人情報を渡してしまうのは嫌だという、謎の嫌悪感(これは大学に入るまで消えなかった)がニコニコ動画を直接見るのを阻んでいました. ニコ動の動画は全てyoutubeへの転載で見ていました. あの頃はニコ動に対してある種手が届かないことによる憧れを抱いていたことでしょう. もっとも、大学生になって初めてニコ動を覗いた時は、検索すらまともにヒットしないゴミツールじゃないかという印象を受けてちょっと幻滅したものですが笑.

# ボカロとの出会い

ボカロを最初に認知したのは、小学生の頃(2008~2009頃でしょうか)、youtubeでトムとジェリーを見てた時期に、見たいストーリーが無くて色々動画を開けてみてた折、ミクをトムとジェリーの動画の一部に登場させてた動画にぶつかったことからですね. トムの絵の上にただ単にミクの絵が半透明に重ねられた"雑"なもので、小5か小6の僕はこれをトムとジェリーへの侮辱だ!と憤ったのを覚えています. そのせいか、名前は覚えていなかったんですけど、ミクのシルエットや、その特徴は、頭に残りました. つまり最初は何者かは知らないが、自分が浸かっているコンテンツを汚す存在だったわけです.

そのあと、ざっくり言うと受験に失敗して、トップで入学した3流校では一時は成績面で栄光を手にしたものの、それを維持できず、発売されたばかりの3DSにハマり、親の目を盗んでゲームに明け暮れ、成績は当然右肩下がり. 向上心は多分にあったものの、気持ち的には暗黒時代に突入しました. そんな状態の中3のある日に、ハマっていたモンハン(Monster Hunter)で自分がプレイできなかったタイトル (MHFです. 当然ネトゲなんて親の許可の問題でできなかったし、当時はネトゲにお金を払うことに価値を見出せなかったので) のあるプレイ動画に、偶然BGMとしてボカロ曲があったわけです. この動画では肉声カバーですが、とても良い曲だったので頑張って原曲や歌い手が歌ったページを探した結果、そこから鏡音リン・レンを知り、初音ミクを知り、世界が一気に開けました. 2013年終わり頃の話です.

# ボカロと学業の成績

時を同じくして、あまり大きくない模試で、初めて学年1位をとって、成績優秀者欄に名前が載りました. これは今思えば本当に偶然で、一つ人生を変えたポイントだったと思いますね. この出来事が引き金になり、ミクを中心にボカロが見せてくれた開かれた世界と、電子が作り出す未来な世界観と希望が、相乗的に良い影響を及ぼしあい、成績は急速に良くなって行きました. 気づけば学年トップの座は不動のものとなり、その後いろいろありながらも、東大にも入ることができました. この、一時はいろんなことがなーなーになっていた自分を、上を目指そうとしても結果が付いてこなかった自分を、その適切な時期に巡り会えて、そこからずっと応援してくれた(と、僕は勝手に思っているわけだけど、)僕にとってのソウルコンテンツは、本当に僕の人生を良い方向に持って行ってくれたと思います.

# 高校時代のボカロとの関わり

ボカロもいわゆるサブカルチャーという括りに入れるならば、中学から高校に上がる時期に、ラノベ、アニメなど、他のサブカルにも芋づる式に引き込まれ、従って自分の中でボカロは相対的に下火になっていきました. (そこで飽きが来たという訳ではないです.) ただ、流すボカロ曲のレパートリーは高3になるまであまり増えなかったように思います. それでも、辛い時や、アニソン以外で聴く曲といえばボカロでした. 特にTell Your Worldは、いつ聴いてもやる気と未来への希望が湧いて出る曲として、受験期間はよく聴いていました.

高校生の間、思い出すようにボカロ今どうなってるかなって調べることがよくあったんですけど、ボカロがどんどん社会に受容されてゆき、カラオケで歌われる曲の上位を占めたりと、もはやサブカルではなくなっていて、一つの音楽のジャンルになっているなと、そう感じました. (もちろん、実際には僕が当時思っている以上にいろんな人が、いろんなやり方でボカロに接していて、もともとサブカルだと自分が思い込んでただけなのかもしれませんが.) それはとてもいいことだと思ったのを覚えているし、一ボカロファンとして誇らしくも思いました. そうしているうちに、僕は大学生になっていました.

# ぱてゼミが僕に与えた影響

僕とボカロとの関係を語る上で、避けては通れない話題の一つでしょう.

# 邂逅

ボーカロイド音楽論(以降ぱてゼミと言います). そんな授業があるのを知ったのは大学に入ってから半年した2016年の秋. シラバスにアホくさ授業があるなぁと. どうせボカロコンテンツを知った風な口して解説するんだろう、然もなくばボカロが音楽界に与えた影響とか論じるんだろうと. 当時の僕は説明も読まず、名前だけで一切を見定め、決める、という、一種の病に陥っていることは確かでした. それが今治ったかと言われればその保証はありません. せいぜい意識を持つことで症状を弱めているだけでしょう. それはさておき、この#東大ぱてゼミがTwitterで盛り上がると、みんな授業の最中楽しそうにボカロを聴いてるではないか!、なんかにゃーん(ぱてゼミのなかで他言無用な話題の総称)な話題がちらほらあって気になる!、と言った具合に、自分がもとより大好きなコンテンツについて、大勢の同じ環境にいる人たちが盛り上がっているのを側から見ているほどもどかしいことはありません. 残念ながらその時間は他の授業を入れてしまったため、抜け出してぱてゼミに行くわけにもいきません. 運が良かったのは、ぱてゼミに延長戦があったことです. しかもその延長戦でゲストまで招待していました! このおかげで僕はようやくぱてゼミの教室に顔を出すことに成功したわけです.

# 初参加

僕にとっての初回講義は、第4回のボーナストラックでした. ゲストが一名いらっしゃっていたのを覚えています. もうどんな話があったかは忘れてしまったんですが、とにかく話し手からも聴き手からもある種の興奮と盛り上がりを感じて、それでいて冷静さと丁寧さを併せ持っていて、また新しい世界を覗いたようでした. これをきっかけに、ぱてゼミの世界に一気に引き込まれていきました.

# ボカロの新たな一面

誤解を恐れずに言えば、今まで僕が聞いてきたボカロ曲はほとんど全て社会における"ポジティブ"な部分の曲だったことに気づかされました. ぱてゼミで扱った曲は、それまで自分が聞きもしない曲ばかりでした. 講演では、その作品の詩、旋律、そして挿絵に至るまで、幅広く作品を鑑賞したことを覚えています. それに加えて、その作品に関連する現実の議題を、とても大胆に、かつ繊細に考察を加えたことも覚えています. ボカロ曲でこのように作品を鑑賞することも、僕が今までしてこなかったことの一つであり、新鮮な経験でした. とは言っても、知識や体験の不足もあり、結局理解できない部分も多くありましたが笑.

# 新たな発見

まず、ボカロに関心がある学生が、こんなに多くいたのかということに驚きました. それまで自分は無意識にボカロという愛好は、圧倒的少数の中にあり、あえて口に出す話題でもないと思い込んでいたのでしょう. これは、高校の頃も大学でぱてゼミと出会うまで、友人ともボカロの話題にほとんど触れなかったからかもしれません.

他方、自分がボカロに関していかにその一部しか齧っていないか、ということを痛感しました. そして、表現意欲がみなぎる大勢の学生を横にしたとき、別段、自分は何もボカロの曲に関して詳しいどころか、その作り方から何まで何も知らなかったわけです. そこで気づいたのは、高校生に入ってから、自分の中でも無意識にボカロが「ただの」音楽の一ジャンルになっていたということです. 講義室にいた他の学生には、ボカロにもっともっと詳しく、情熱を注ぎ、自ら積極的に作品を制作するような人が大勢いました. 東大に入ってから、様々な自信を削がれてきた僕ですが、そのような人たちを目の当たりにした僕は、ボカロには詳しいぞと言うアイデンティティまで見失ってしまいました. つまり、ボカロとの関係は人それぞれとは言うものの、僕はそこでもう一度ボーカロイドと自分の関係を、他人と比較した上で見つめ直す作業に迫られたわけです.

ボーカロイド音楽論を通じて色々な曲に出会え、色々な考え方を巡り歩き、色々な人と知り合うことができました. 時には自分の耳に痛いことも聞かされたけれども、それこそぱてゼミのスパイスであり、それを受けてどう反応しようと、いい刺激だったと思います.

# ソウルコンテンツのいま

ぱてゼミで知った曲や、自分でネットサーフィンして良いと思った曲も含めて、大学でもそれなりに新しいボカロの曲を仕入れてきました. 中学・高校の初めあたりから間が開いたせいか、僕の知るPたちは、ある人は長期間活動を休止し、ある人は地声に乗り換え、ある人はもう新曲を聴きたくても聞けないといった状況になっていたりしていました. ボーカロイドを取り巻く環境は確かに僕が知る環境から大きな変化があり、もちろん好きな曲もたくさん見つけられたけれど、その環境の変化には、浦島気分でもあったし、一抹の寂しさを覚えたものです.

僕がハマりだした2013年冬の時期は、ボカロの人気が今振り返ってみればピークに当たる時期で、人気曲がアップロードされて何ヶ月や何年も経っていなかったり、オンタイムなものが多かったように思います. しかし、多くの記事がデータを持ち出して指摘するように、そのコンテンツ規模は、ニコニコ動画の勢いが衰えるのと同じくして、以前より小さくなっていることは、間違いないでしょう. この状況が続き、これから僕より下の世代がボカロを聞かなくなっていくというのは、考えてみれば寂しく感じてしまってやみません. でも、たとえボカロがというコンテンツ自体がこれからどんどん細くなっていくとしても、ボカロを心に持っている人が消えることはなく、そのような人と、あの頃はこんなものがあったよなと話せることは、それはそれでとても嬉しいことでなないでしょうか.

ある意味で、ボーカロイドという商品は、一つの時代を象徴する技術だったわけですから、新しい技術に取って代わられることは、それも仕方がないように思います. ボーカロイドが、その技術をして人々に夢を与えるという役割の第一線を他の何かに譲り渡したとしても、ボーカロイド自体が消えるというわけではなく、その記録には数々の人の軌跡が刻まれているわけで、なおかつこの一つの時代を象徴するような存在として、歴史に残ることでしょう. だから、僕はこのコンテンツが将来どのような発展を遂げるにせよ、悲観的な見方はしていません.

このように書いてしまうと、僕が上のように言えるのはボーカロイドに付与されたキャラクターをあまり重要視していない、などと思われるかもしれませんが、これに関しては否定しようにも強く否定はできないような気がします. 冷静に考えれば、ぼくがこのコンテンツで一番興奮を覚えたのは、一人の歌姫という、(あえて言うならば)「題材」を元に、作り、集まり、それが新たな網や輪を作り、それが繋がり、広がっていく、という流れでした. 僕はボカロに、人間の素晴らしさを見たのかもしれません. この点において、僕は、もしかしたらそのキャラクターについてはあまり重要視していないのでしょう. そうはいっても、クリプトンキャラも、その他のキャラも、大好きです笑.

# 未来を魅せてくれた存在から

2013年、僕が中学生の頃ボカロにハマったとき、いくつかの点で大きな感動と憧れと高揚感を抱いたんですが、それはおそらく以下のようなことだったと思います.

  • 各クリエイターが描く、それぞれのボカロキャラクターを中心とする独特の世界観に魅了された
  • ボカロ自体やミクのライブなどに、(社会に陽に普及していないという意味での)先進テクノロジーが多用されいて、頭に思い描く近未来像をそのまま顕現したように感じられた
  • インターネット上で多くの人が、曲を作る人もただ楽しむ人も、そのコンテンツを一丸になって盛り上げようという一種の熱を感じ取れ、自分もその中にいると思えた

多分、これらは他の人と同じで、いたって普通な理由だと思います. 正直なことを言うと、一番ボカロにハマっていたのは中学から高校にかけての時期で、現在のボカロの流れについてはすっかり疎くなってしまいました. これからも、何か良い曲ないかなぁと、たまに戻ってくるだけになるかもしれません. これまでは戻ってくるたびに、古い良い曲があり、新しい良い曲に痺れを繰り返したわけですが、願わくばこのサイクルはずっと続けばなぁと、思っている次第です.

変わらないもの

  • ボカロに出会えてよかった
  • ボカロが成長する自分を後押ししてくれた
  • ボカロが僕が世界を一つ広げるきっかけを作ってくれた

という感情はおそらく一生変わることはないでしょう.

ボーカロイドに最初に出会った時の感動は今も鮮明に覚えています. それが自分の世界観の大きな構成要素になっていることは間違いありません. ボーカロイドが誰かの心にとまり、それがきっかけになり、新しい、人々を興奮させるような、誰かの心に刻まれるような何かを生み出していくならば、それはとても素敵なことではないでしょうか. そうすれば、またたくさんの点は線になって、いくつもの線は円になって、全て繋げてく、どこにだって、ですね!
