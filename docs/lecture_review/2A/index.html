  • 評価方法 : (課題 + ) 期末試験 ( + 出席は救済用)

# 講義

宮野悟先生のテキストに沿って、蓮尾先生が解説を加えていく講義形式で、オートマトン(DFA, NFA、PDA など)、正則文法、正則表現、最小オートマトン、文脈自由文法などを扱います. 予習をしてから臨むようにというのが授業のスタイルで、予習でわからなかった部分などを含め、授業中に随時質問を受け付けているタイプの先生です.

先生の話し方がメリハリあって眠くならないです. たまにジョークも混ぜてきます. とは言っても僕は真面目に勉強していなかったので、授業もわからないことだらけだし、今になって授業で覚えているのは、「計算理論は有限と無限のせめぎ合い」「構文論と意味論は違う」「帰納法を回せないと研究所をクビになる」という3つだけです笑.

2019年の講義からは先生が変更になって、宮尾先生が担当することになりました.

宮尾先生は自然言語処理の先生で、温厚な話し方が特徴です.
