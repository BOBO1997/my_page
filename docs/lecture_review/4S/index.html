# 幾何学XC

基本情報

  • 開講日時 : 金1
  • 先生         : 松尾 厚 先生
  • 評価方法 : 期末試験
  • 備考         :

# 講義

前半6回で集合と位相の必要な部分を講義します. ここの部分はペースが非常にゆっくりで、このペースでド・ラームコホモロジーまでたどり着くのか心配なくらいでした. 後半6回は、多様体の定義から始め、接ベクトル空間、接束、などを紹介し、結局ドラームコホモロジーまで無理やりたどり着かせました. 案の定、後半になるにつれ、爆速で進んでいきました. 回によっては延長5分/延長10分なども.

# 試験

集合位相の問題が2問、普遍性の問題が1問、多様体関連の問題が2問と言ったところです. 難しかった... 試験で点を取ろうと思うならば、集合位相の演習問題に慣れておくのが良さそうです.
