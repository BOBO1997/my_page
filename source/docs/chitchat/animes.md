# アニメの感想とか

::: tip 並び順
- アルファベット順です.
- 漢字、仮名文字で始まる作品は、ローマ字読みにした時のアルファベット順にしています.
- 数字で始まる作品はその正式な読み方の最初の発音のアルファベット順です.
- 2期以上ある作品は同じ項目で扱います.
:::

::: tip 
最近忙しくて、アニメなかなか観る時間が取れてないけど、オススメのアニメがあれば、いつでも教えてください.
:::

::: tip 概要の並び順
- 原作
- アニメ放送日時
- アニメーション制作会社
- 個人的な評価/オススメの度合い
:::

## 劇場版オリジナル

::: tip
アニメがあって、劇場版にもなっているものは、アニメの方で言及します.
:::

### 君の名は

いいぞこれ.

### HELLO WORLD

一行瑠璃さんが可愛かった. 本作最大の見所(僕得).

PR動画は"恋愛要素を前面に出して"、ヒロインを救いに行くというものだったけど、実際は映画内で与えられたその他の情報量が多すぎて、恋愛要素が二の次になっているし、そもそも世界設定に対する疑問が多すぎて映画に集中できなくなった.
前半はこれから面白そうなことが起こるという予感がしながら映画を見ていたけど、後半は情報量の過多に加え、急展開すぎて冷めてしまった.

#### 絵について

完全CGなのかな?
エフェクト描写や、町の描写などはとても良かった.
人の表情の動きはやっぱりCGには難しいみたい.
そこを割り切れば、作画についてはとてもいいと思う.

#### 声について

主人公もヒロインも、声がぎこちない. (というか、感情を付けるシーンで割と棒読みに近い話し方をしてる)

#### 解釈

よくわからないというのが正直なところ.

ネタバレになるので、[こちらのページ](/chitchat/animes/hello_world)へ
